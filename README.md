# SkillMatch — Job Portal UI

A React + Vite frontend for a job portal concept, built to practice component-based UI design and client-side state management.

## Features
- Home, Jobs, Companies, About pages with client-side navigation (via useState)
- Login and Signup forms (UI only — not yet connected to a backend)
- Responsive job listing and company cards

## Tech Stack
- React 19
- Vite
- Vanilla CSS

## Status
This is a frontend-only prototype. Authentication and job data are currently hardcoded/simulated — no backend or database is connected yet.

## Run locally
```
npm install
npm run dev
```
